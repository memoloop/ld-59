# PROPAGATION

This is the prototype of the game created for Ludum Dare 59.
It's a simple puzzle platformer where you play as a robot navigating a maze with various walls, levers, obstacles, and signal receivers...

## Inputs
- A - move left
- D - move right
- SPACE - jump
- E - interact
- Q - propagate signal

## What did I use?
- Game Engine: Godot 4.6.2
- Code Editor: Visual Studio Code
- Sprites: LibreSprite 1.1-dev
- Musics and sfx: LMMS
