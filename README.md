# PROPAGATION

This is an originally a prototype of a game created for Ludum Dare 59.
It's a simple puzzle platformer where you play as a robot navigating a maze with various walls, levers, obstacles, and signal receivers...

I am still working on this project. There will be more levels and mechanics...

## Inputs
- A - move left
- D - move right
- SPACE - jump
- E - interact
- Q - propagate signal
- F11 or ALT+ENTER - toggle fullscreen
- ESC or P - pause

The inputs can be changed in settings

## What did I use?
- Game Engine: Godot 4.6.2
- Code Editor: Visual Studio Code
- Sprites: Pixelorama, LibreSprite 1.1-dev (old)
- Musics and sfx: LMMS
